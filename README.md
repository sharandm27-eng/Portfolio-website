# D.M. Sharan — Portfolio Website

An interactive, animation-driven portfolio built to showcase real-world projects, skills, and experience — designed with a bold dark-red aesthetic, custom cursor effects, and smooth page transitions.

**Live Site:** https://portfolio-website-one-beta-32.vercel.app/

---

## ✦ Overview

This is a personal portfolio site for **D.M. Sharan**, an aspiring Full Stack & AI Developer currently pursuing B.Tech in Computer Science (AI & ML). The site highlights real, shipped projects — including an AI-powered health management system, a cross-platform Flutter marketplace app, and a role-based campus management platform.

Rather than a static resume-in-a-browser, this site focuses on **motion and interactivity** — every section transition, cursor movement, and text reveal is custom-built to feel distinct from a typical template portfolio.

---

## ✦ Features

- **Custom storm-generator cursor** — a themed animated cursor that follows mouse movement across the entire site
- **Lightning strike page-transition effect** on the landing page for a strong first-impression entrance
- **Scramble/decode text animations** on section headings for a futuristic, terminal-style reveal
- **Blend-mode typography** — large background name text merges visually with the hero portrait using CSS blend modes
- **Expandable project cards** — click to reveal descriptions, tech stack, and live/code links without leaving the page
- **Fully responsive** — optimized layouts across desktop and mobile
- **Role-based, real-world projects** — not templated placeholders; every listed project is live and functional

---

## ✦ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS, custom CSS (glassmorphism, blend modes)
- **Animation:** Framer Motion / GSAP, custom Canvas-based cursor & particle effects
- **Routing:** TanStack Router
- **Deployment:** Vercel

## ✦ Getting Started

Clone and run the project locally:

```bash
git clone https://github.com/sharandm27-eng/Portfolio-website.git
cd Portfolio-website
npm install
npm run dev
```

The site will be available at `http://localhost:5173`.
